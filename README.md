# LLM Platform Analytics

## Project Overview

This project analyzes enterprise-style telemetry data from a Large Language Model (LLM) platform to understand user engagement, product adoption, operational performance, and operational costs.

## Domain

AI & LLM Usage Analytics

## Industry

AI & LLM Platforms

## Problem Statement

Analyze user engagement, product adoption, and operational performance across an LLM platform to identify opportunities for improving user retention, increasing product adoption, optimizing operational efficiency, and reducing platform costs.

## Dataset

**LLM System Ops Production Telemetry Dataset**

Source: Hugging Face
